Project

MIDI Home Control Rig

Turning an Akai MPK Mini into a physical smart-home remote: gesture logic → MQTT → Home Assistant, with a live web UI for visibility and feedback.

Highlights

  • Real-time MIDI input routing with short/long/hold behaviors
  • Config-driven mapping layer for pads/keys/knobs
  • Low-latency MQTT integration with Home Assistant automations
  • Live Web UI: layout, mappings, event stream, and interaction feedback

Stack

Python MIDI MQTT Home Assistant WebSockets HTML/CSS/JS

Everything is built to be low-maintenance and easy to extend.

Overview

How it works, at a high level.

A Python router listens to MIDI input and classifies gestures (tap/hold/knob changes). It publishes compact MQTT messages. Home Assistant subscribes and performs the actual device control. A small web UI consumes event updates to show what’s happening and provide immediate visual feedback.
